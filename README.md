# 🛡️ Cybersecurity Dashboard & Monitoring System

[![React](https://img.shields.io/badge/React-19-blue.svg?logo=react)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-8-646CFF.svg?logo=vite)](https://vitejs.dev/)
[![Netlify](https://img.shields.io/badge/Netlify-Deployed-00AD9F.svg?logo=netlify)](https://lrtimp.netlify.app/)

Современное SPA-приложение для мониторинга безопасности электронных сервисов. Обладает уникальной визуализацией в виде **интерактивного щита**, системой управления событиями и ролевой моделью доступа.

---

## ✨ Ключевые особенности

*   **🛰️ Интерактивный щит (SVG):** Визуализация уровня защищенности сервисов в реальном времени с поддержкой наведения и анимации.
*   **🔐 Система безопасности:** Ролевая модель доступа (Администратор/Пользователь), блокировка учетных записей и аутентификация.
*   **📊 Журнал событий:** Логирование попыток входа, подозрительной активности и изменений в системе.
*   **🛠️ Управление защитой:** Применение точечных мер безопасности (Firewall, SSL, 2FA) для повышения уровня защиты сервисов.
*   **🎨 Премиальный дизайн:** Тёмная тема с неоновыми акцентами, Glassmorphism эффекты и микро-анимации.

## 🚀 Технологический стек

*   **Frontend:** React 19, Vite, React Router DOM v7.
*   **Стилизация:** Vanilla CSS + CSS Modules.
*   **API:** Axios (интеграция с MockAPI и json-server).
*   **Развертывание:** Netlify (CI/CD через GitHub).

## ⚙️ Запуск и Разработка

### 1. Подготовка
```bash
# Клонирование репозитория
git clone https://github.com/Varkym/LR1_TIMP.git
cd LR1_TIMP

# Установка зависимостей
npm install
```

### 2. Запуск локально
```bash
# Запуск режима разработки
npm run dev
```

### 3. Сборка для продакшена
```bash
npm run build
```

## ☁️ Интеграция с Backend (MockAPI)

Приложение настроено на работу с **MockAPI** через единый ресурс `api`, что позволяет имитировать работу с базой данных в облаке. 

**Основные эндпоинты:**
*   `GET /api` — Получение всех данных системы.
*   `PUT /api/:id` — Синхронизация измененного состояния.
